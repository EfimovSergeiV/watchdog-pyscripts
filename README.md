### Installation

###### Windows

```zsh
pip install pyaudio numpy opencv-python
.\myenv\Scripts\activate
```

###### GNU/Linux

```zsh
sudo apt install python3-pyaudio portaudio19-dev python3-all-dev
pip install pyaudio
```
